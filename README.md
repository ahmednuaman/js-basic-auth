# Usage
1. Run `npm install`
2. Run `npm start`
3. Visit [http://localhost:8000/index.html](http://localhost:8000/index.html)
4. Enter incorrect details of `bar, bar`, it fails
5. Enter the correct details of `foo, bar`, it passes
6. Profit
